Planifyy - Твоят интелигентен органайзер

Идея:
Planifyy е мобилно приложение за управление на задачи, 
създадено да помогне на потребителите да организират ежедневието си 
лесно и ефективно. Целта е бързо добавяне, преглед и проследяване на ангажименти.

Как работи:
Приложението използва локална SQLite база данни за сигурно съхранение на информацията.
Потребителският интерфейс комуникира с базата чрез DatabaseHelper клас, който изпълнява 
CRUD операции. Когато добавиш задача, тя се записва с време и дата, като броячът 
в реално време обновява статистиката. Логиката е проектирана да бъде лека и 
бърза, като автоматично подрежда задачите според времето на изпълнение чрез SQL заявки.

Архитектура:
Model-View с директен достъп до данни (DatabaseHelper), осигурява лесна поддръжка 
и тестване на кода.

Потребителски поток:
Начален екран: Потребителят вижда списък с всички добавени задачи.
Добавяне: С натискане на бутона "+" се отваря форма за въвеждане на заглавие, час и дата.
Запис: След потвърждение задачата се записва в базата и се появява в списъка.
Статистика: Общият брой задачи се обновява автоматично при всяка промяна.

Технологии и версии:
Kotlin - 1.9+
Android SDK - 34
SQLite - Local
JUnit/Robolectric - 4.13 / 4.11

Стъпки за стартиране:
Изтегли репозиториумa.
Отвори проекта в Android Studio.
Синхронизирай Gradle файловете.
Изгради проекта (Build -> Rebuild Project)
Стартирай приложението на емулатор с API 34.

Тестови акаунти:
Приложението работи локално и не изисква потребителски акаунти или автентикация в облак.
